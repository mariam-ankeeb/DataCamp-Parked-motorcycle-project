# Parked motorcycle Project (Net Revenue Analysis)

##  Overview
This project analyzes wholesale sales data to calculate net revenue per product line, per month, per warehouse.  
The goal was to extract business insights using SQL.

##  Tools & Technologies
- PostgreSQL
- SQL (Aggregation, CASE, GROUP BY)
- DataCamp (for learning & practice)

##  Key Concepts Practiced
- Filtering data with `WHERE`
- Grouping data with `GROUP BY`
- Date extraction using `EXTRACT()`
- Calculating net revenue using `SUM(total) - SUM(payment_fee)`
- Sorting results with `ORDER BY`

##  Insight
Net revenue values help understand which product lines and warehouses generate the most revenue during June–August.

##  Dataset
- product_line  
- warehouse  
- date  
- total  
- payment_fee  
- client_type
